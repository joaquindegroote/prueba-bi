# Prueba BI Demo

Esta app Streamlit muestra la resolución de la prueba técnica de BI/DS.

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Estructura

```
data/                 # CSV de ejemplo
streamlit_app.py      # App Streamlit
requirements.txt      # Dependencias
```

La misma versión está desplegada en Streamlit Community Cloud.
